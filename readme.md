# Tube Clone

Tube Clone is a clone of [https://www.bigduckgames.com/flowfree](https://www.bigduckgames.com/flowfree) written is cpp as a kata.

![screnshot](./screenshot.png "screen shot")

# Build requirements

- cmake
- conan
- gcc

# How to build

    mkdir build && cd build
    conan install ..
    cmake ..
    make
